VISIT SITE

CartoCMS

Designer, co-author

CartoCMS is a custom content management system designed and developed for New Cartographers and built using Ruby on Rails. I handled the UI/UX design, while Brian McNabb handled the code.
A reaction against the often cumbersome pre-fab CMS's on the market today, CartoCMS is a streamlined set of tools that we customize to suit the specific needs of each client.

Among other things, the CMS features content archiving, batch asset uploading, dynamic page creation / templating, in-page editing, multiple user accounts, and user tracking / statistics.
